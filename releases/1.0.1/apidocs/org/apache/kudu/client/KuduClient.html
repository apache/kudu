org.apache.kudu.client

Class KuduClient

  • java.lang.Object
    • org.apache.kudu.client.KuduClient
  • All Implemented Interfaces:
    AutoCloseable

    @InterfaceAudience.Public
@InterfaceStability.Evolving
public class KuduClient
extends Object
implements AutoCloseable
    A synchronous and thread-safe client for Kudu.

    This class acts as a wrapper around AsyncKuduClient. The Deferred objects are joined against using the default admin operation timeout (see KuduClient.KuduClientBuilder.defaultAdminOperationTimeoutMs(long) (long)}).
